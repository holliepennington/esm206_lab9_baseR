#lets combine all of the above

We will use ‘numeric’ to create an empty vector we will use ‘for’ to iterate along items of that empty vector we will use ‘[]’ to access each a lot and fill-in that empty vector we will use ‘rnorm’ to sample values from a random normal distribution we will plot the trend

\[N_{t}=N_{t-1} + r\left(1 - \frac{N_{t-1}}{K}\right)\]

\[ r \sim norm(\mu = 1.5, \sigma = 0.8) \] ## writing the code down

# set up model parameters 
K <- 100 
nsteps <- 100
mu <- 1.5
sigma <- 0.8 

# set up model objects 
N <- vector(mode = "numeric", length = nsteps)
N[1] <- 25

# model it 
for(t in 2:nsteps){
  r <- rnorm(n = 1, mean = mu, sd = sigma)
  N[t] <- N[t-1] + r * (1 - (N[t-1]/K))
}

#visualize it 
plot(N)

Something More

We will use ‘matrix’ to create an empty matrix (rows are time, columns are runs) we will repeat the process

# set up model parameters 
K <- 100 
nsteps <- 100
mu <- 0.4
sigma <- 1 

# set up model objects 
N <- matrix(data = NA, nrow = nsteps, ncol = 100)
N[1,] <- 25

# model it 
for(t in 2:nsteps){
  r <- rnorm(n = 100, mean = mu, sd = sigma)
  N[t,] <- N[t-1,] + r * (1 - (N[t-1,]/K))
}

#visualize it 
matplot(N, type = "l")
